Workflow to get the data in vcf file:

To process each vcf

1 - Sort vcf by position
grep '^#' slim_output_t1_1.vcf > slim_output_t1_1_sorted.vcf && grep -v '^#' slim_output_t1_1.vcf | sort -k1,1 -k2,2n >> slim_output_t1_1_sorted.vcf

2 - bgzip the sorted file
bgzip slim_output_t1_1_sorted.vcf

3 - tabix the bgziped file
tabix slim_output_t1_1_sorted.vcf.gz

4 - Extract the information 
bcftools query -f 'chr%CHROM\t%POS\t%MT\tY\t%REF,%ALT[\t%GT\t]\t%MID\t%DOM\t%S\t%PO\t%GO\t%AC\n' slim_output_t1_1_sorted.vcf.gz

To merge and process the merged vcf

1 - Sort vcf by position
grep '^#' slim_output_t1_1.vcf > slim_output_t1_1_sorted.vcf && grep -v '^#' slim_output_t1_1.vcf | sort -k1,1 -k2,2n >> slim_output_t1_1_sorted.vcf

grep '^#' slim_output_t2_1.vcf > slim_output_t2_1_sorted.vcf && grep -v '^#' slim_output_t2_1.vcf | sort -k1,1 -k2,2n >> slim_output_t2_1_sorted.vcf

2 - bgzip the sorted file
bgzip slim_output_t1_1_sorted.vcf

bgzip slim_output_t2_1_sorted.vcf

3 - tabix the bgziped file
tabix slim_output_t1_1_sorted.vcf.gz

tabix slim_output_t2_1_sorted.vcf.gz

4 - merge vcf_t1.vcf and vcf_t2.vcf files
bcftools merge --force-samples  slim_output_t1_1_sorted.vcf.gz slim_output_t2_1_sorted.vcf.gz -o slim_output_merged_1.vcf

5 - Extract the information 
bcftools query -f 'chr%CHROM\t%POS\t%MT\tY\t%REF,%ALT[\t%GT\t]\t%MID\t%DOM\t%S\t%PO\t%GO\t%AC\n' slim_output_merged_1.vcf > slim_output_merged_1.txt


final vcf output:
1 - sorted mutations by positions;
2 - ./. for genotypes of mutations that aren't present in one of the files;
3 - Allele count AC is a sum of the allele counts in each vcf
